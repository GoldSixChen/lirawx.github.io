java基础知识笔记（2）

May 20, 2017

java

接口(interface)

  1. 接口不能用于实例化对象
  2. 接口没有构造函数
  3. 接口的所有方法必须是抽象方法
  4. 接口不能包括成员变量，除 static final 修饰外
  5. 接口不能被类继承，只能被类实现
  6. 接口可以多重继承

    接口的方法只能是 public abstract 修饰，变量只能public static final修饰
