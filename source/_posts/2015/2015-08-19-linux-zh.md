###首先安装fbterm
```
sudo  apt-get install fbterm
```


###然后把你的用户加入video用户组
```
sudo gpasswd -a xi video
```
原因是这普通用户(xi)不能启动fbterm，现在基本上就可以显示了，还可以输入中文，但我觉得太鸡肋。
