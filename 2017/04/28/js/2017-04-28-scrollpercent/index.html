javascript滑动条（程序代码）

2017-04-28

拖拽效果的实现，主要是利用，鼠标按下不起来的事件中加入鼠标的移动事件来实现，
所以当鼠标按下的时候，记录位置，然后移动鼠标，改变位置，
window.getSelection ? window.getSelection().removeAllRanges() : document.selection.empty();代码的作用清楚所选的内容
然后当鼠标弹起的时候，不做操作.
